Installation
===========================================
.. _install:

RAW can be downloaded from `sourceforge <https://sourceforge.net/projects/bioxtasraw/>`_.
There are prebuilt installers for windows (.msi) and mac (.dmg). Installation on Linux requires
building from source.

RAW is also available through `SBGrid <https://sbgrid.org/software/titles/bioxtas-raw>`_.

Some features in RAW require a separate installation of the
`ATSAS <https://www.embl-hamburg.de/biosaxs/software.html>`_ package.

Some features in RAW require a separate installation of the
`EMAN2 <http://blake.bcm.edu/emanwiki/EMAN2/Install>`_ package.
