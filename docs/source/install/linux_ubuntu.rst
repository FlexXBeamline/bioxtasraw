Ubuntu/Linux Mint/Debian install from source instructions
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
.. _lnxubuntu:

Tested on Ubuntu 16.04, Linux Mint 18.3, and Debian 9.4.

First install the g++ compiler:

#.  Open a terminal (right click on the desktop and select :menuselection:`New Terminal`
    or :menuselection:`Open in terminal`) and run the following commands (hit
    enter/return to execute each command):

    *   ``su`` (if necessary)
    *   ``sudo apt-get update``
    *   ``sudo apt-get install g++``
